# quickask

QuickAsk - Mobile-first async questionnaire tool

## Stack
Next.js (App Router) with Tailwind CSS and TypeScript.

## Rules
- Run `npm install` before doing anything else
- Keep dependencies minimal — this runs on a 4GB VPS
- Use App Router (`app/` directory), not Pages Router
- Tailwind for styling, no extra CSS-in-JS libraries
- Stay within this project directory
- Commit working code only
